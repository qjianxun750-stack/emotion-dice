# 🎲 情绪骰子 - Emotion Dice

一个轻量级的情绪骰子H5网页应用，快速验证市场的产品MVP版本。

## ✨ 功能特性

- 🎯 **8套骰子主题**：状态、发疯、祈愿、玄学、关系、人生、妈妈、搭子
- 🎲 **3D翻转动画**：炫酷的骰子旋转效果
- 📸 **分享图生成**：一键生成精美分享图片
- 📝 **历史记录**：自动保存最近的摇骰记录
- 📱 **响应式设计**：完美适配手机和电脑
- 👆 **手势操作**：左右滑动快速切换骰子主题
- 💾 **本地存储**：无需后端，数据保存在浏览器

## 🚀 快速开始

### 方式一：直接打开（最简单）

1. 直接双击 `index.html` 文件
2. 在浏览器中即可使用

### 方式二：本地服务器（推荐）

```bash
# 使用Python快速启动
cd emotion-dice
python -m http.server 8000

# 或使用Node.js
npx serve
```

然后访问：`http://localhost:8000`

## 🌐 免费部署到网络

### 部署到 Vercel（推荐）

1. 注册 [Vercel](https://vercel.com)
2. 安装 Vercel CLI：`npm i -g vercel`
3. 在项目目录执行：`vercel`
4. 按提示完成部署
5. 获得免费域名：`your-project.vercel.app`

### 部署到 GitHub Pages

1. 创建GitHub仓库
2. 上传项目文件
3. 在仓库设置中启用 GitHub Pages
4. 获得免费域名：`yourusername.github.io/emotion-dice`

## 🎨 自定义骰子内容

编辑 `js/app.js` 文件中的 `DICE_CONFIG` 数组：

```javascript
const DICE_CONFIG = [
    {
        id: 'status',
        name: '今日状态骰',
        color: '#FF6B6B',  // 主题色
        faces: [
            { 
                word: '小爷来了',      // 显示文字
                emoji: '😎',           // 表情符号
                desc: '今天有气势，冲！' // 详细说明
            },
            // ... 其他5个面
        ]
    },
    // ... 其他骰子主题
];
```

修改后刷新页面即可生效！

## 🎯 使用方式

1. **选择骰子主题**：点击顶部标签或左右滑动切换
2. **摇骰子**：点击骰子或点击"点击骰子摇一摇"
3. **查看结果**：等待2秒动画结束后显示结果
4. **生成分享图**：点击"生成分享图"按钮
5. **保存图片**：在预览窗口点击"保存图片"
