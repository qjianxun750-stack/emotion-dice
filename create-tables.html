🗄️ Supabase 数据表SQL语句

一键复制所有SQL，到Supabase控制台执行即可

📋 操作步骤：

  1. 访问 https://supabase.com/dashboard
  2. 选择项目 emotion-dice-stats
  3. 点击左侧菜单的 SQL Editor（SQL图标）
  4. 点击 New query 按钮
  5. 点击下面的"复制SQL"按钮，粘贴到编辑器
  6. 点击 Run 或按 Ctrl+Enter
  7. 看到 Success 就成功了！
📊 表1：visitors（访客表） 
CREATE TABLE IF NOT EXISTS visitors (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    visitor_id TEXT NOT NULL,
    first_visit TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
    last_visit TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
    device_type TEXT,
    UNIQUE(visitor_id)
);
🎲 表2：dice_rolls（骰子记录表） 
CREATE TABLE IF NOT EXISTS dice_rolls (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    visitor_id TEXT NOT NULL,
    dice_name TEXT NOT NULL,
    dice_id TEXT NOT NULL,
    result_word TEXT NOT NULL,
    result_emoji TEXT NOT NULL,
    result_desc TEXT,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
📤 表3：shares（分享记录表） 
CREATE TABLE IF NOT EXISTS shares (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    visitor_id TEXT NOT NULL,
    share_type TEXT NOT NULL,
    dice_name TEXT,
    result_word TEXT,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

✅ 创建完成后

点击左侧菜单的 Table Editor，应该能看到3个表：visitors、dice_rolls、shares

然后告诉我"表已创建"，统计功能就能正常工作了！
