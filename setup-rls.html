🔧 Supabase RLS 策略配置工具

📋 配置说明 此工具将帮助你在 Supabase 中配置 RLS 策略，允许匿名用户读写数据。
只需 3 个简单步骤，大约需要 2 分钟。
1
打开 Supabase SQL Editor
1. 访问 https://supabase.com/dashboard
2. 选择项目 emotion-dice-stats
3. 左侧菜单点击 SQL Editor
4. 点击 New query 创建新查询
2
复制 SQL 命令
点击下面的按钮复制 SQL 命令：
✅ 已复制！ 
-- ========== 允许匿名用户写入数据 ==========
-- 1. 允许匿名用户插入访客数据
CREATE POLICY "Enable insert for anonymous"
ON visitors
FOR INSERT
TO anon
WITH CHECK (true);

-- 2. 允许匿名用户插入摇骰子数据
CREATE POLICY "Enable insert for anonymous"
ON dice_rolls
FOR INSERT
TO anon
WITH CHECK (true);

-- 3. 允许匿名用户插入分享数据
CREATE POLICY "Enable insert for anonymous"
ON shares
FOR INSERT
TO anon
WITH CHECK (true);

-- 4. 允许匿名用户读取访客数据
CREATE POLICY "Enable select for anonymous"
ON visitors
FOR SELECT
TO anon
USING (true);

-- 5. 允许匿名用户读取摇骰子数据
CREATE POLICY "Enable select for anonymous"
ON dice_rolls
FOR SELECT
TO anon
USING (true);

-- 6. 允许匿名用户读取分享数据
CREATE POLICY "Enable select for anonymous"
ON shares
FOR SELECT
TO anon
USING (true);
3
执行 SQL 命令
1. 在 SQL Editor 中粘贴复制的命令
2. 点击右下角的 Run 按钮
3. 看到 "Success" 提示表示配置成功
