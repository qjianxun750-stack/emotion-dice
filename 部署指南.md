# 📱 部署到手机访问 - 简单指南

## 方法一：Netlify Drop（最简单，推荐⭐⭐⭐⭐⭐）

### 步骤：

1. **打开网页**
   ```
   https://app.netlify.com/drop
   ```

2. **找到项目文件夹**
   ```
   d:\我的AI世界\01知识库\精选知识库\04、我的实战\视频带货\骰子\emotion-dice
   ```

3. **拖拽上传**
   - 把整个 `emotion-dice` 文件夹拖到网页上
   - 等待10-30秒

4. **获得链接**
   - 部署成功后会显示链接
   - 类似：`https://emotion-dice-abc123.netlify.app`

5. **手机访问**
   - 复制链接发送到手机
   - 在手机浏览器打开即可

## 方法二：Vercel 命令行部署

### 前提条件：
需要先注册 Vercel 账号：https://vercel.com

### 部署命令：
```bash
cd "d:\我的AI世界\01知识库\精选知识库\04、我的实战\视频带货\骰子\emotion-dice"
npx vercel
```

按提示操作：
- Set up and deploy? 输入 `Y`
- Which scope? 选择你的账号
- Link to existing project? 输入 `N`
- Project name? 直接回车（使用默认）
- Directory? 直接回车

等待部署完成，会显示链接。

---

## 方法三：GitHub Pages（免费永久）

### 步骤：

1. **创建 GitHub 仓库**
   - 访问 https://github.com
   - 创建新仓库，命名为 `emotion-dice`
   - 勾选 "Add a README file"

2. **上传文件**
   - 点击 "Add file" → "Upload files"
   - 上传所有文件（保持文件夹结构）
   - 点击 "Commit changes"

3. **启用 GitHub Pages**
   - 进入 Settings → Pages
   - Source 选择 "main" 分支
   - 点击 Save

4. **获得链接**
   ```
   https://你的用户名.github.io/emotion-dice
   ```
